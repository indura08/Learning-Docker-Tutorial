Docker Inspect: An Overview
===========================
docker inspect is a Docker command that provides detailed information about Docker objects, such as containers, images, volumes, or networks. It returns information in JSON format, making it easy to read and understand.

Purpose of docker inspect
--------------------------

1.Detailed Information: Get comprehensive details about Docker objects.

2.Debugging: Useful for diagnosing issues with containers and images.

3.Configuration Check: Verify configurations and settings of Docker objects.
